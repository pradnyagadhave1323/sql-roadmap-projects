## 🚀 How to Use These Files

You can run any phase's SQL files in:
- **MySQL Workbench**
- **pgAdmin** (PostgreSQL)
- **DBeaver**
- **SQLiteOnline.com** (browser-based, no install needed)

**Recommended order per phase:**
```
1. schema.sql   → Creates the tables
2. seed.sql     → Populates with sample data
3. queries.sql  → Run and study the queries
```
